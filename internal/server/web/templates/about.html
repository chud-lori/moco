About

This is a personal shelf, shared with you.

Moco is a small self-hosted reader and publisher for books, papers, and longer writing. You can upload PDF, EPUB, or Markdown, keep it private to yourself, share by email, or publish a copy to the public shelf where anyone can read it in their browser — no account needed.

Who runs this one

This instance is run by Lori. Everything on the public shelf is something I read, wrote, or wanted to share. There's no algorithm and no ranking — the order is just "what I published most recently." If a book ends up here, it's because I thought it was worth your time.

Why I built it

I read a lot of PDFs, EPUBs, and rough Markdown drafts. Kindle is fine for novels; Calibre is fine for organizing; neither is great for the in-between case — papers, zines, draft chapters of things friends are working on. Moco is what I wanted: a reader that runs in a browser, syncs progress and highlights across devices, and optionally lets me publish a copy with one toggle.

How to follow

When I publish something new to the public shelf, it shows up in the Atom feed. Drop that URL into any RSS reader (NetNewsWire, Reeder, Inoreader, Feedly — they all accept Atom) and you'll see new books as I publish them. No email signup, no mailing list.

Run your own

Moco is open source. If you want a personal shelf like this one for yourself, the code and setup notes live on GitHub. It's a single Go binary with SQLite — runs anywhere Docker runs.
